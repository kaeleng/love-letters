Dear GitHub,

I am sorry. I have been using you wrong for a very long time. But I appreciate you being patient with me as I have learned how to properly use you. Because of you, I am able to store my git repositories online and share my code with others. All I have to do is go to github.com and create a free account and then create a new repository. I can use the terminal to cd into my local git repository project by using git remote add origin github-repo-url and git push -u origin master. Once I am really ready to commit to this, I can simply git push. I can also make a README to add more information about the committed files. I feel so safe with you, as I know my files will never get lost once they are pushed to GitHub. Thank you for the security.

Love, Kaelen
